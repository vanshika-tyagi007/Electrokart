# ⚡ ElectroKart

A modern full-stack e-commerce platform for purchasing electronic products online. ElectroKart provides a responsive shopping experience with secure authentication, product browsing, shopping cart management, order placement, and an admin dashboard for managing the store.

## 🌐 Live Demo

https://electrokart-dye6.onrender.com/

---

# 📌 Features

### Customer Features

- User Registration & Login
- Secure Authentication
- Browse Products
- Product Categories
- Search Products
- Product Details Page
- Add to Cart
- Update Cart Quantity
- Remove Items from Cart
- Wishlist Support
- Checkout Process
- Order History
- Responsive Design
- User Profile Management

### Admin Features

- Admin Authentication
- Dashboard
- Add Products
- Edit Products
- Delete Products
- Manage Categories
- Manage Orders
- Manage Users
- Inventory Management
